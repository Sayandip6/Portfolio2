# Sayandip Naskar Portfolio

A modern, interactive portfolio built with Next.js to showcase my work as a Full Stack MERN developer. The site combines smooth motion, bold visuals, and a clean information structure to present projects, skills, and experience in a professional format.

## Live Demo

- Production site: [https://arashi-pf.netlify.app/](https://arashi-pf.netlify.app/)
- LinkedIn: [https://www.linkedin.com/in/sayandip-naskar-9a4031375/](https://www.linkedin.com/in/sayandip-naskar-9a4031375/)

## Highlights

- Responsive single-page portfolio with a polished hero section and smooth motion.
- Sections for about, skills, projects, experience, and contact.
- Built with a modern app-router structure and optimized for production.
- Designed to communicate both technical depth and visual craft.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## Available Scripts

- `npm run dev` - start the development server
- `npm run build` - create a production build
- `npm run start` - run the production server
- `npm run lint` - run ESLint
