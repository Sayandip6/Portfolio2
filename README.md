## 🚀 Getting Started

### Prerequisites
- **Node.js** 18+ 
- **npm** or **yarn** package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Sayandip6/Portfolio.git
   cd Portfolio/portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```

## 🎨 Customization

### Edit Content

**Quick find all editable sections:**
- Search for `// EDIT:` in `.tsx` and `.css` files

### Key Sections to Customize

| Section | File | Line |
|---------|------|------|
| Navigation Items | `page.tsx` | ~24 |
| Skills & Stack | `page.tsx` | ~28 |
| Projects | `page.tsx` | ~47 |
| Experience/Timeline | `page.tsx` | ~68 |
| Footer Text | `page.tsx` | ~452 |
| Site Title & Description | `layout.tsx` | ~14-18 |
| Theme Colors | `globals.css` | ~4-5 |
| Cursor Style | `globals.css` | ~20, 26, 30 |

### Changing Icons
All icons from [Lucide React](https://lucide.dev) - modify imports in `page.tsx` lines 4-19

### Theme Colors
Edit CSS variables in `globals.css`:
```css
:root {
  --background: #020617;    /* Dark background */
  --foreground: #f8fafc;    /* Light text */
}
```

## 🌐 Deployment

### Deploy on Vercel (Recommended)

1. **Push to GitHub**
   ```bash
   git add .
   git commit -m "Initial commit"
   git push origin main
   ```

2. **Connect to Vercel**
   - Visit [vercel.com](https://vercel.com)
   - Click "New Project" → Import your GitHub repository
   - Configure environment variables (if any)
   - Deploy!

3. **Your live URL:**
   ```
   https://your-project-name.vercel.app
   ```

### Other Deployment Options
- **Netlify** - Connect GitHub, auto-deploy on push
- **GitHub Pages** - Static site hosting
- **AWS Amplify** - Full-stack hosting with backend support
